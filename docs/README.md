## Directory Structure

```
/sanskargupta551.github.io/
│
├── _config.yml               # Jekyll configuration (site metadata, navigation, etc.)
├── index.md (or index.html)  # Home page content
│
├── _projects/                # Folder for project content (as collection)
│    ├── project1.md
│    ├── project2.md
│    ├── project3.md
│    └── ...                  # Add more for each project
│
├── _layouts/
│    ├── default.html         # Base layout for site pages
│    ├── home.html            # Home page layout (optional)
│    └── project.html         # Layout specifically for projects
│
├── _includes/                # Reusable HTML snippets (header, footer, nav)
│
├── assets/
│    ├── css/                 # Stylesheets
│    ├── js/                  # JavaScript files
│    └── images/              # Site and project images
│
├── about.md                  # About or Contact page
└── README.md                 # Info about the repo
```
